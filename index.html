Symphony Data Acquisition System

Symphony is a data acquisition system focused on electrophysiology experiments. Symphony allows research scientists to write well-structured, user-configurable acquisition routines in MATLAB without getting caught up in a lot of non-essential details. It also provides an easy-to-use graphical user interface for running those routines as part of an overall, well-documented experiment.

Requirements

  • HEKA ITC18 (PCI or USB interface)
  • Axon MultiClamp 700B
  • Windows 7 (32 or 64-bit)
  • MATLAB 2012b or later (32-bit only)

Installation

  • Download the latest release.
  • Run the included setup.exe.

Documentation is available on the project wiki.
